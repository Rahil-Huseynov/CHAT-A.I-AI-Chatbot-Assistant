# 💬 CHAT-A.I — AI Chatbot Assistant

A modern chatbot UI built with **React + TypeScript + Vite**.

---

## 🚀 Getting Started

Follow the steps below to run the project locally.

### 1. Clone the repository

```bash
git clone https://github.com/Rahil-Huseynov/CHAT-A.I-AI-Chatbot-Assistant.git
cd CHAT-A.I-AI-Chatbot-Assistant
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run development server

```bash
npm run dev
```

---

## 🌐 Open in Browser

```
http://localhost:5173
```

---

## ⚙️ Requirements

* Node.js (v18 or higher)
* npm / yarn / pnpm
